## Rules of use

1. Use only what's marked **public** in the "Status" line per client.
   If a client is marked **NDA — confirm with Noel**, don't publish
   their name or numbers without explicit approval.
2. Use exact numbers as written. Don't round, don't combine, don't
   estimate.
3. Where a number has a `// TODO verify` flag, ask Noel before using
   it on a public page.
4. Where a date range is approximate (e.g. "circa 2018-2022"), use
   it carefully. Don't cite as a specific date.
5. Cross-reference testimonials in `_docs/references/testimonials.md`.
